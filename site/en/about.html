About TWGEX

TWGEX is a Taiwan-equities market-structure observatory. We organise public data — the index and turnover, option gamma exposure (GEX), institutional flow direction, volatility and a 0–10 structural temperature — into a few legible charts, so you can read the market's current state at a glance, like a weather report. It provides information and an observation framework, not investment advice.

About the developer

TWGEX is built and maintained by an independent developer. Taiwan's market-structure data — institutional flows, option gamma positioning, volatility — is scattered and hard to read. TWGEX organises this public data into a few legible charts, so you can read the market's current structural state at a glance, like a weather report. It provides information and an observation framework, not investment advice, and never covers individual stocks.

Data sources

The underlying market data used on this site comes from:

  • Index, turnover, institutional spot net buy/sell: Taiwan Stock Exchange (TWSE)
  • TAIEX option open interest and quotes, institutional futures/option positions, margin balances: Taiwan Futures Exchange (TAIFEX)

The above public data is used under the Open Government Data License, Taiwan, version 1.0 (OGDL v1.0), with attribution to the source agencies. The GEX, smart-money direction, temperature and other indicators shown here are derived computations based on that public data; their methodology is described in the definition blocks on each page.
